# Ψ-дуальные числа (Psi-Dual Numbers)

[![Python](https://img.shields.io/badge/Python-3.7+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

Иерархическое обобщение дуальных чисел для вычисления производных высших порядков методом автоматического дифференцирования.

## Описание

Проект реализует новую алгебраическую систему — **$\Psi$-дуальные числа**, которые позволяют эффективно вычислять производные любого натурального порядка функции одной переменной.

### Особенности

- **Прямое извлечение производных** — производные извлекаются напрямую из коэффициентов разложения
- **Иерархия нильпотентов** — система элементов $\psi_k$, где \[ψ_{k+1} = \sqrt{ψ_k}\]
- **Вычисление до порядка 2ᵐ** — вычисление $f(x + \psi_m)$ содержит все производные до порядка $2^m$
- **Сложность $\mathcal{O}(N^2)$** — асимптотически оптимальный метод forward-mode AD

## Математическая основа

### Ψ-дуальные числа

$\Psi$-дуальное число имеет вид:
z = a + $\Psi$,
где $\Psi$ — линейная комбинация $\psi$-компонент с различными весами.

### Иерархия ψ-компонент

Базисные элементы ψₓ удовлетворяют аксиомам:
- $\psi_x = 0$ при $x < 0$
- $\psi_0 \ne 0$ и $\psi_0 ^ 2 = 0$ (классический нильпотент)
- $\psi_{x+1} = \sqrt{\psi_x}$ (иерархическое соотношение)
